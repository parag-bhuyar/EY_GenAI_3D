  BEFORE THE SESSION — do the two "ollama pull" downloads in advance.
  Together they are about 1.6 GB. Doing them at the start of class, on the
  same network as everyone else, will cost you 15+ minutes of the lab.

  Requires Ollama installed first: https://ollama.com/download
  Check it works:  ollama --version


  Mac/Linux:
  cd rag-project
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ollama pull nomic-embed-text
  ollama pull llama3.2:1b
  streamlit run app.py

  Windows:
  cd rag-project
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  ollama pull nomic-embed-text
  ollama pull llama3.2:1b
  streamlit run app.py

  Then open http://localhost:8501.


  BOTH models are required:
    nomic-embed-text  (~274 MB)  turns your PDF chunks into vectors
    llama3.2:1b       (~1.3 GB)  writes the answer from the retrieved chunks

  If you pull only the first one, indexing will work and then answering
  will fail. That is the most common way to get stuck in this lab.

  No API key and no cost — both models run on your own machine.
